Pick the Color and Keep It

The world of color is amazing and beautiful, however all the varieties of colors will inevitably fade away with time going by. The color picking bee intends to gather colors from objects and keep them in a honeycomb-like interactive light installation.

Interaction

First, let the bee picks up the color by color sensor and then the light inside it shows the same color. When it is moved close enough to the honeycomb, the photoresistor in the honeycomb can sense the light, so the color signals will be tranfered to the honeycomb by wireless transmission modules. Then the light in the honeycomb turns on and reflects the same color. Meanwhile the light in the bee gradually fades away.
